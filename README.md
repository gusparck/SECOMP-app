# Sistema de agendamento - SECOMP
O SECOMP-app é um sistema mobile estruturado dentro dos padrões de desenvolvimento web para otimizar e centralizar a gestão da Semana da Computação do DECSI, na UFOP. Ele tem o objetivo de melhorar a experiência de alunos, organizadores e palestrantes, oferecendo ferramentas diretas para inscrições em atividades, check-in, interação em tempo real e notificações.
